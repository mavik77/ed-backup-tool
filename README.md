# Elite Dangerous Backup Tool

GUI tool to back up:
- Journal (logs)
- Bindings
- Graphics

Each selected category is exported into a separate ZIP file.

## Run from source
pip install -r requirements.txt
python ed_backup_gui.py

## Disclaimer
Unofficial fan-made utility. "Elite Dangerous" is a trademark of Frontier Developments.

## Portable (Windows)
Download the ZIP from Releases, extract, run `EDBackupTool.exe`.
Do not remove the `_internal` folder.
